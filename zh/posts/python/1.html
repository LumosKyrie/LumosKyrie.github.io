                      # 入门书籍推荐

                      # 1.Python语言

                      Python由荷兰数学和计算机科学研究学会的吉多·范罗苏姆于1990年代初设计，作为一门叫做ABC语言的替代品。Python提供了高效的高级数据结构，还能简单有效地面向对象编程。Python语法和动态类型，以及解释型语言的本质，使它成为多数平台上写脚和快速开发应用的编程语言，随着版本的不断更新和语言新功能的添加，逐渐被用于独立的、大型项目的开发。

                      python可以说是大数据，机器学习，深度学习的首选语言，得天独厚的解释性语言优势在python中可谓体现的淋漓尽致，各种封装，各种包，import一下解决其他语言几十行干的事，让编程变得如此简单！！

                      所谓机器学习，其实就是计算机对一部分数据进行学习，然后对另外一些数据进行预测与判断。机器学习的核心是“使用算法解析数据，从中学习，然后对新数据做出决定或预测”。也就是说计算机利用以获取的数据得出某一模型，然后利用此模型进行预测的一种方法，这个过程跟人的学习过程有些类似，比如人获取一定的经验，可以对新问题进行预测。

                      😉人身苦短，我用python！

                      # 2.IDE

                      工欲善其事，必先利其器！说到python的IDE就不得不提到以下三种：

                      （1）Pycharm（强烈推荐）☀️

                      Pycharm是一个多功能的IDE，尤其是代码补全和调试器十分出色。Pycharm适合使用Python开发中型或大型系统。这款IDE也是我用来写python脚本用的最多的IDE，有很多的主题可以选择，并且配合ananonda可以配置多个环境。通常情况下，不同的项目需要配置不同的开发环境，项目太多怎么办呢？把原有的环境不停的安装、还原？显然，这样的做法显然不可取。

                      事实上，我们在开发的过程中通常会使用ananonda为每一个项目配置一个虚拟环境，在不同虚拟环境env下，有各自项目对应的python版本、包的版本。好处在于打开不同的项目时及时切换对应的环境、这样就不会与主机本身的环境发生冲突，导致系统奔溃等后果（惨痛教训，过来人强烈推荐，一定要学会使用ananonda）。

                      Jetbrains提供两种Pycharm版本，一种是专业版(订阅模式)，另一种是社区版(免费模式)。在大多数情况下，社区版足以进行开发。但是，我个人比较推荐使用专业版，专业版的功能更丰富，主题也多。问题来了，如何白嫖专业版呢？

                      正儿八经的白嫖方式：大家去校园邮箱注册网站，申请一个校园邮箱账号（edu.cn），然后使用校园邮箱申请免费使用专业版，但是有时间限制。

                      其他方法：当然也有魔法可以破解，有需要的同学可以联系我。

                      🔥刚入门的同学可能会把一堆包、各种版本的python，tensorflow...安装在主机环境，这种做法极不可取，因为你会出现各种让人抓狂的Bug，特别是编程基础薄弱的同学，非常容易出现一杯茶，一包烟，一个bug调一天的情况。

                      （2）Visual Studio Code

                      宇宙第一IDE，不解释。开发人员可以通过安装一些插件的Visual Studio Code在各种编程环境中进行开发。 vs code提供Python插件，配置Python环境十分容易，但是要配置调式工具可能你会多花一点时间。

                      （3）Jupyter Notebook

                      Jupyter Notebook适用于数据可视化或机器学习，因为它在以单元格为单位的代码执行和绘制图形和表格方面表现出色。可以实时显示plt的图像，挺好的。

                      # 3.书籍推荐

                      （1）《Python编程：从入门到实践》

                      👍这本书可以说是新手入门必备的书籍，书上的案例十分适合0基础的同学。

                      该书的优点

                      简单、通俗、围绕python的基础语法，而且不仅仅局限于基础知识的介绍，它还能让你学习开发功能齐全的项目，书中讲解了如何开发三个项目，包括简单的Python 2D 游戏开发如何利用数据生成交互式的信息图，以及创建和定制简单的Web 应用。建议大家将书上的例程在自己的IDE里实现一下，重点熟悉数据结构、算法封装、怎么写接口、调用、类等。

                      该书的缺点

                      有些包（模块）、指令、可能版本有些老旧、用现有的python版本可能会出现bug。至于书本后边的几个项目，大家粗略看看就行，熟悉一下web的框架，表单的提取，发送方式等就行了。

                      至于想要开发项目的话，建议大家在B站上搜索一下Flask，或者Django的项目动手做一做就行了。这里我比较推荐大家从flask学起，我们一般做个简单的web系统，flask框架足够，至于Django的话我推荐大家观看尚硅谷的谷粒商城项目

                      （2）《Python机器学习基础教程》

                      👍学完python的基础语法，其实已经足够应对机器学习的基础算法了，接下来就是机器学习入门的必读书籍《Python机器学习基础教程》。这本书的主要内容包括：机器学习的基本概念及其应用；实践中最常用的机器学习算法以及这些算法的优缺点；在机器学习中待处理数据的呈现方式的重要性，以及应重点关注数据的哪些方面；模型评估和调参的高级方法，重点讲解交叉验证和网格搜索；管道的概念；如何将前面各章的方法应用到文本数据上，还介绍了一些文本特有的处理方法。

                      这本书是我反复阅读的书籍，使我受益匪浅！此外，它还对我的论文发表起到十分大的帮助！想要做机器学习，深度学习的同学，希望大家能把它认真看完，动手把书上的实例过一遍。重点是理解，切记不要急功近利闷头敲，要把书上的一些数学知识也适当的消化，一些机器学习的专业用语也要熟知。

                      （3）《统计学方法》

                      👍李航博士所著的《统计学方法》可以说是机器学习的入门宝典了。第二版在第一版的基础上增加了无监督学习的内容，更加丰富，是非常值得学习的材料，配合《Python机器学习基础教程》能让你对算法背后的数学逻辑有一个清晰的认识。书上的许多案例，都有详细的解题过程，对初学者十分友好，此外简单的计算可以让你的思维更上一层楼，是进行算法创新的基石。重点在于理解机器学习算法背后的理论基础，包括熵的概念，随机森林的剪枝，概率论，梯度提升等。建议大家把书上的一些算法，动手计算一下。

                      机器学习本质是统计学规律的算法表现，要想面对审稿人的问题，要从这些底层的逻辑、过程是必要的。

                      （4）《利用Python进行数据分析》

                      👍本书由pandas项目创始人Wes McKinney亲笔撰写，详细介绍利用Python进行操作、处理、清洗和规整数据等方面的具体细节和基本要点。第2版针对Python 3.6进行全面修订和更新，涵盖新版的pandas、NumPy、IPython和Jupyter，并增加大量实际案例，可以帮助你高效解决一系列数据分析问题。

                      Pandas、NumPy是你必须要掌握的俩个模块，如何处理Raw data，是进行机器学习的第一步，首先得把输入数据的type，shape搞懂，因为很多同学在编写代码的过程中都会出现类别不平衡，广播，缺失行，类型出错，维度等bug，这些问题的根源其实还是你数据预处理过程中，没有把输入和输出的shape，读写数据搞清楚。

                      （5）《机器学习》

                      西瓜书（《机器学习》，清华大学出版社）和花书（《深度学习》，人民邮电出版社）分别是目前国内机器学习、深度学习领域销量最大的教材。这本书上手有难度，特别是繁杂的公式，涉及面太广，没有深厚的数学功底读起来很困难。尤其是不少数学知识超出了本科的“微积分”，“线性代数”，“概率论与数理统计”的范围。一些复杂的算法，比如支持向量机、反向传播算法、EM算法、概率图模型、变分推断，让你看完之后脑子里只剩懵逼俩字。

                      但是，B站有个东北姐姐讲的挺不错的，推荐大家结合视频来学，事倍功半。此外，还有一本南瓜书是对西瓜书数学公式推导的注解，也挺好的。

                      （6）《推荐系统算法实践》《推荐系统开发实战》

                      《推荐系统算法实践》主要讲解推荐系统中的召回算法和排序算法，以及各个算法在主流工具Sklearn、Spark、TensorFlow等中的实现和应用。讲解推荐系统中的排序算法，包括线性模型、树模型和深度学习模型，分别介绍逻辑回归、FM、决策树、随机森林、GBDT、GBDT+LR、集成学习、深度森林、DNN、Wide & Deep、DeepFM、YouTube推荐等模型的原理，以及其在Sklearn、Spark、TensorFlow主流工具中的实现与应用。

                      《推荐系统开发实战》是一本关于推荐系统从入门到进阶的读物，采用“理论+实践”的形式展开，不仅对各种推荐算法进行了介绍，还对算法所涉及的基础理论知识进行了补充。

                      这俩本书只需要粗略的阅读，书籍中的大多数算法都是在电商平台，谷歌浏览器等大公司中应用的算法，相对前边的几本书，算法比较新，但是复现有一定的难度，因为他们的数据量庞大，计算环境大多是spark等分布式、云平台，我们只需要通过CSDN、Github等搜索一下相对简单的例程即可，重点是挖掘创新点。

                      读完这几本书，基本上已经上道了，如果想继续深入的学习，推荐大家学习深度学习的一部分经典读物，我在这里就不推荐了，因为多数的深度学习大多是用于图像的处理，这里涉及的内容十分庞大。
